# Задача № 1
Создать класс Student\
Поля класса (Значения нужно передать через конструктор):
* Имя студента
* Фамилия студента
* Группа
* Факультет
* Курс
* Возраст

Методы класса:
* Info(); (Вывести данные в консоль столбиком) 
Transfer(); (Перевод студента в другую группу и факультет)
* если значение пустое вывести ошибку в консоль
